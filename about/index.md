<center><a href="http://taylantatli.github.io/Moon"><b>Moon</b></a> is a minimal, one column jekyll theme.</center>

## Features
* algorithm: X11
* ticker: AXE
* prefix: P
* supply: 21000000
* maturity: 101 blocks
* confirmations: 6
* spacing: 2.5
* diff: DGW
* port: 9937
* rpc port: 9337
